# Сравниваем вакансии программистов
Скрипт запрашивает с двух ресурсов Headhunter и SuperJob вакансии программистов по нескольким популярным языкам программирования.

Для каждой вакансии считает примерную заработную плату, если она указана. На основе этого, проверяя все вакансии для каждого яызка программирования считает среднюю заработную плату по каждому из них.

Выводит таблицу для каждого ресурса в которой указаны: языки программирования, сколько всего найдено вакансий для каждого языка, сколько вакансии было учтено при подсчете средней заработной платы и среднюю заработную плату.

## Как установить
Для установки потребуется серкретный ключ приложения SuperJob, получить его можно зарегистрировав приложение по следующей ссылке https://api.superjob.ru/.

Полученный ключ присвоить переменной `SJ_SECRET_KEY` в файле `.env`, как в примере ниже:

```
SJ_SECRET_KEY=elrsjhnvtow3874y65tvnoae8794485739084759
```

Для запуска скрипта использовать команду:

```
python main.py
```

Python3 должен быть уже установлен. Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```
